**FRAUDPULSE:**

FraudPulse is an end-to-end credit card fraud detection ecosystem that bridges the gap between machine learning and human investigation. It features a high-frequency "Forensic Lab" that visualizes transaction telemetry in real-time, allowing security analysts to detect, inspect, and neutralize fraudulent clusters across geographic nodes.

**Key Features:**

◉ Vector Stream Monitoring: A live-refreshing scatter plot using logarithmic scaling to visualize transaction density and risk simultaneously.

◉ Biometric User Inspector: A "Click-to-Inspect" panel that allows deep-dives into specific User IDs, transaction categories, and forensic scores.

◉ Dynamic Risk Velocity: A real-time intelligence header that calculates "Fraud Speed" and "Active Threat Counts" every 2 seconds.

◉ Geospatial Intelligence: Integrated Power BI reporting for high-level geographic heat maps and long-term auditing.

**Technical Stack:**

◉ Backend & AI: Python 3.11, XGBoost (Pattern Recognition Engine)

◉ Database: PostgreSQL (High-Concurrency Data Hub)

◉ Forensic UI: Dash by Plotly (Web Command Center)

◉ Data Processing: Pandas, NumPy, SQLAlchemy

◉ Analytics: Power BI (Executive Reporting)
